# 台股加減倉邏輯框架：5 個市場訊號詳解 (量化實作版)

> 參考美股量化框架，針對台股「淺碟型經濟」、「外資主導權高」、「半導體權重極高」與「散戶參與度高」之特性進行本土化改良，並解決邏輯矛盾，提供具體實作數據源。

---

## 一、總結

台股作為新興市場，極易受到全球資金流向與半導體景氣循環影響。本框架篩選出 5 個完全公開且免費的台股關鍵指標：台指 VIX、融資維持率與外資籌碼、櫃買指數（OTC）與市場廣度、新台幣匯率（資金流）、以及跨資產聯動（費半與美元）。

籌碼面訊號已於 2026-05 擴充，新增**外資現股買賣超**與**投信現股買賣超**（來源：TWSE BFI82U），以補足原先僅追蹤外資期貨部位的盲點——期貨反映避險意圖，現股才真正決定指數漲跌。

透過綜合分析這些指標，投資者可以理性判斷台股的回檔是「健康的加倉機會」還是「外資撤資的系統性危機」，從而制定分批建倉或防禦策略，克服追高殺低的心理障礙。

## 二、各訊號詳細說明與資料來源

### 訊號一：台指波動率指數（TAIEX VIX）
*   **定義：** 反映市場機構對未來 30 天台股大盤波動的預期（期權保險費）。
*   **實作修正（解決回落矛盾）：** VIX 衝高時代表恐慌，但**「正在飆升的 VIX」是飛刀，不能接；「從高點回落的 VIX」才是底部確認**。因此，評分邏輯將引入「VIX 趨勢過濾器」（詳見第五章評分表）。
*   **⚠️ 實作代理說明（方案 B）：** 台指 VIX 原始資料**目前無任何公開 API 可取得**——`^TVIX` 為美股已下市 ETN，期交所（TAIFEX）歷史資料頁面已改版為軟 404。本框架改用 **台灣加權指數（^TWII）之 20 日年化歷史波動率（HV20）** 作為代理，HV20 趨勢方向與 VIX 高度一致，絕對數值偏低約 5–10%，故門檻做相應校正（見第五章）。計算公式：前 20 個交易日對數報酬之標準差 × √252 × 100%。
*   **資料來源：**
    *   **Yahoo Finance Ticker：** `^TWII`（計算 HV20，不再需要 VIX 專用 Ticker）

### 訊號二：籌碼與情緒指標（融資 ＋ 融券 ＋ 外資期權）
*   **定義：** 結合「散戶槓桿（融資維持率）」、「空方籌碼（融券）」與「外資態度（台指期未平倉）」。
*   **數據統一：** 本框架統一將**外資期貨淨空單警戒門檻設定為 30,000 口**。
*   **資料來源：**
    *   **融資維持率 ＋ 融券餘額（2026-06 新增融券）：**
        *   **來源：** 台灣證券交易所（TWSE）每日收盤後（約 18:00）公布之「信用交易統計」。
        *   **公開 API：** `https://www.twse.com.tw/rwd/zh/marginTrading/MI_MARGN?date=YYYYMMDD&response=json`
        *   **⚠️ 融資實作代理說明（方案 B）：** 此 API **僅提供融資金額（仟元）餘額**，無法直接計算維持率百分比。本框架改用**融資金額每日變動幅度**作為代理指標（大幅下降 → 斷頭壓力出現；持續上升 → 槓桿過高）。因係代理指標，給分**減半**以反映近似誤差（見第五章）。
        *   **融券餘額（逆向指標）：** 同一支 API 的 `tables[1]` 即為融券統計。融券5日增幅 > 3% 代表空方過度擠擁，是短線偏多的逆向指標（潛在軋空動能）。注意：此訊號為逆向指標，需搭配其他訊號共振，不能單獨判讀。
    *   **外資期貨未平倉口數：**
        *   **來源：** 台灣期貨交易所（TAIFEX）每日下午 15:00 公布之「三大法人期貨部位」。
        *   **公開 API / 爬蟲路徑：** `https://www.taifex.com.tw/cht/3/futContractsDate`（POST 請求，參數 `queryDate=YYYY/MM/DD`，免登入，可直接爬取外資臺股期貨淨部位）。
    *   **外資現股買賣超 ＋ 投信現股買賣超（2026-05 新增）：**
        *   **來源：** 台灣證券交易所（TWSE）每日收盤後公布之「三大法人買賣金額統計表」。
        *   **公開 API：** `https://www.twse.com.tw/rwd/zh/fund/BFI82U?response=json`（無需參數，回傳本月每日明細，欄位 index 3 = 外資差額、index 9 = 投信差額，單位：元）。
        *   **為何補充現股？** 外資期貨反映「避險 / 方向性賭注」，而現股買賣超才是實際資金流入流出的直接證據，對指數漲跌的解釋力更強。連3日現股買超代表外資真正進場，而非只是期貨多翻空的假訊號。
        *   **⚠️ 資料時效說明：** BFI82U 於當日收盤後更新，盤中查詢時最新一筆為**前一個交易日**數字。
        *   **⚠️ 自營商排除說明：** BFI82U 雖含自營商資料，但自營商的股票部位大量來自認購權證的 delta-hedge（被動對沖），並非方向性判斷，故不納入評分邏輯。

### 訊號三：市場廣度（Market Breadth）
*   **定義：** 觀察台股上漲是靠「台積電獨撐（拉積盤）」還是「多數股票普漲」。本框架使用兩層廣度指標互補：OTC vs 加權（中小型股vs大型股）+ 台積電相對強弱（最大權值股方向確認）。
*   **資料來源：**
    *   **加權指數 (TAIEX) Ticker：** `^TWII` (Yahoo Finance)
    *   **櫃買指數 (OTC) Ticker：** `^TWOII` (Yahoo Finance)
    *   **台積電 Ticker：** `2330.TW` (Yahoo Finance)（2026-06 新增）
*   **台積電相對強弱說明（2026-06 新增）：**
    *   台積電佔加權指數約 **30%** 市值權重，是台股最重要的單一指標股。
    *   **近5日跑贏大盤 > 2%** → +1：台積電確認領漲，半導體循環訊號健康，整體加權上漲具說服力。
    *   **近5日落後大盤 > 2%** → -1：台積電轉弱，即使 OTC 廣度看起來健康，加權的漲幅也難以持續；或大盤在跌而台積電跌更深，結構更為危險。
    *   **⚠️ 與 OTC 廣度的互補性：** OTC 反映中小型股，台積電反映半導體龍頭，兩者視角不同。最理想狀況是兩者都偏正面；最危險是兩者均偏負面（拉積盤 + 台積電轉弱）。

## 三、核心觀點

1.  **台股是「台積電的武林」：** 判斷台股不能只看加權指數。若忽視 OTC 指數與市場廣度，容易陷入「賺了指數、賠了差價」或「指數沒跌但個股崩盤」的陷阱。
2.  **匯率是台股的血液：** 只要新台幣沒有出現失控性急貶，台股的下跌多屬技術面或估值修正；一旦台幣失控貶值，代表外資無情提款，此時必須嚴格執行防禦。
3.  **散戶斷頭是最佳買點（3-3-4 策略）：** 當融資維持率跌破 140% 甚至逼近 130% 時，市場會出現非理性的多殺多與斷頭賣壓。此時沒人能精準預測最低點，建議採用 **「3-3-4 分批建倉策略」** 克服心魔：
    *   **第一批 (30% 資金)：** 當指標初次轉佳（總分達 +1 至 +2）時小量試水。
    *   **第二批 (30% 資金)：** 當市場進一步下探，引發恐慌且指標共振（總分達 $\ge$ +3）時加碼。
    *   **最後一批 (40% 資金)：** 等待關鍵恐慌指標（如 VIX 指數）確認從高點回落，或新台幣止貶轉升時，補入最後主力資金。

## 四、5 個關鍵問答

### Q1：為什麼台股不用美股的「高收益債利差」來判斷信用危機？
台灣的公司債與高收益債市場規模較小、流動性低，無法即時反映市場信用風險。對台股而言，**「新台幣匯率」**就是最靈敏的資金流動性指標，外資撤資（台幣貶值）即等同於信用市場惡化。

### Q2：什麼是「拉積盤」？對操作有何啟示？
「拉積盤」是指資金集中拉抬台積電（2330）等超大型權值股，使加權指數看起來很漂亮，但中小型股（OTC）卻紛紛下跌。這代表市場廣度極差，通常是波段過熱或主力拉高出貨的警訊。

### Q3：融資維持率如何影響台股走勢？
融資是散戶借錢買股的指標。當融資維持率降至 140% 以下，代表許多散戶面臨追繳壓力；一旦跌破 130%，券商將強制斷頭賣出。這種「多殺多」會造成股價超跌，但也因此能快速清空浮額，形成市場底部。

### Q4：台指 VIX 和美股 VIX 有何不同？
台指 VIX 的波動通常比美股 VIX 稍大。在台股，VIX 跌破 13 通常代表市場過度樂觀（過熱）；而當 VIX 飆破 30 時，代表市場進入極度恐慌，此時**應密切觀察 VIX 是否止漲並開始回落**。唯有 VIX 確認從高點回落，才是安全的逆向進場訊號，切忌在 VIX 急衝時盲目接飛刀。

### Q5：當台股過熱（VIX < 13，融資維持率 > 165%）時，應該怎麼做？
此時市場情緒極度自滿，利多已被充分定價。投資人應停止追高，並適度降低融資等槓桿倍數，逢高分批獲利了結漲幅已高的中小型投機股，保留部分現金。

## 六、宏觀背景指標（月更，不影響短線評分）

以下三個指標為月更資料，**不計入評分**，僅作為景氣方向的背景參考。

| 指標 | 說明 | 更新頻率 | 查詢來源 |
| :--- | :--- | :--- | :--- |
| **景氣對策信號燈** | 紅→過熱、黃→趨緩、藍→衰退，由9個子指標合成 | 每月中旬 | 國發會景氣指標查詢系統 |
| **台灣製造業 PMI** | 採購經理人指數，>50 擴張 / <50 收縮，對台股有 1–2 個月領先性 | 每月第一個工作日 | 中華採購與供應管理協會（NISM） |
| **外銷訂單年增率（電子類）** | 台灣電子/半導體外銷訂單，領先出貨 1–2 個月，是半導體庫存循環的早期訊號 | 每月20日前後 | 經濟部統計處 |

> **為何不納入評分？** 以上均為落後指標，市場通常比景氣燈**早見底 3–6 個月**。若直接納入評分，在市場底部（景氣最悲觀時）反而會錯誤壓制加倉訊號。

## 七、宏觀與短線訊號背離解讀

當儀表板的「短線評分」與「宏觀背景」出現背離時，正確解讀如下：

### 核心原則

> **宏觀不告訴你「什麼時候買」，而是告訴你「這次下跌可能有多深、需要多久復原」。**

宏觀景氣收縮期間出現短線加倉訊號，不代表訊號是假的；而是代表這次回調的幅度和持續時間可能超過景氣擴張期的修正，需要更高的帳面虧損容忍度。

### 四種情境對照

| 宏觀狀態 | 短線評分 | 正確解讀 | 操作調整 |
| :--- | :--- | :--- | :--- |
| **樂觀**（綠燈、PMI > 50） | ✅ 加倉訊號 | 最佳買點。牛市回調，訊號最可靠 | 積極執行 3-3-4 策略，不需特別保守 |
| **悲觀**（藍燈、PMI < 50） | ✅ 加倉訊號 | 機會是真實的。但宏觀在底部最悲觀，落後市場 3–6 個月，不代表不能買 | 可進場，但心理上準備等待更長的復甦週期，帳面浮虧容忍度需提高 |
| **樂觀** | 🟠 謹慎訊號 | 牛市中的健康修正，通常不會惡化成熊市 | 可比平時更積極，縮短觀望時間 |
| **悲觀** | 🟠 謹慎訊號 | 雙重警示，下跌可能持續更久 | 嚴守防禦，縮短操作週期，不持有長線部位 |

### 最常見的誤判場景

**「景氣藍燈 ＋ 短線 +3」** — 這往往是市場底部附近的典型型態。

* 景氣燈剛轉藍，代表市場已跌了 3–6 個月，正在或接近見底
* 短線訊號出現加倉，是真實的底部確認
* **錯誤做法**：因為景氣燈藍就不敢買，等到景氣燈轉綠才進場（那時市場已漲了 30%）
* **正確做法**：執行 3-3-4 策略進場，但心理準備「可能還要再等半年才看到大幅獲利」
